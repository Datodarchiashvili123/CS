# კომპიუტერი ნულიდან · Computer From Zero

ინტერაქტიული კურსი ქართულად, რომელიც ასწავლის, **როგორ მუშაობს კომპიუტერი ნულიდან** — ელექტრონებიდან პროგრამის შესრულებამდე. თითო გაკვეთილს აქვს ახსნა, ცოცხალი სიმულაცია და პატარა დავალება.

An interactive, dependency-free course (in Georgian) that builds a computer's ideas from the ground up: electrons → bits → transistors → logic gates → adders → memory → ALU → CPU → program → cache.

## 🚀 გაშვება

ქეშ-გამორთული ლოკალური სერვერით (რეკომენდებული — ცვლილებები მაშინვე ჩანს):

```bash
python3 serve.py
# გახსენი: http://127.0.0.1:8000
```

ან უბრალოდ გახსენი `index.html` ბრაუზერში.

## 📚 გაკვეთილები

1. ელექტრობა — დენი და ელექტრონები
2. ორობითი სისტემა — 0/1, ბიტები, 2-ის ხარისხები
3. ტრანზისტორი — ელექტრონული ჩამრთველი
4. NOT კარიბჭე — ტრანზისტორ + წინაღობა (ჰორიზონტალური სქემა)
5. AND კარიბჭე — სერიული ტრანზისტორები
6. OR კარიბჭე — პარალელური ტრანზისტორები
7. XOR კარიბჭე — მავთულებით შეერთებული კარიბჭული სქემა
8. ნახევრად შემკრები (Half Adder)
9. სრული შემკრები (Full Adder)
10. მეხსიერება — როგორ ინახება ბიტი (ტრიგერი)
11. ALU — არითმეტიკულ-ლოგიკური ბლოკი
12. CPU-ის საფუძვლები — datapath
13. საათი და პროგრამა — fetch → decode → execute
14. ქეშ მეხსიერება — L1 / L2 / L3
15. რეკაპი — მთელი კომპიუტერის რუკა

## ✨ ფუნქციები

- **პროგრესის შენახვა** (localStorage) + გვერდითი ჩამონათვალი checkmark-ებით და პროგრეს-ბარით
- **Deep-linking** — თითო გაკვეთილს აქვს URL (`#alu`, `#cpu`…)
- **კლავიატურით ნავიგაცია** — ← / →
- **მუქი თემა** — ავტომატურად სისტემის მიხედვით (`prefers-color-scheme`)
- **ნამდვილი სქემატური სიმბოლოები** — VDD, GND, წინაღობა, MOSFET, ლოგიკური კარიბჭეები — SVG წრედებით
- **ხელმისაწვდომობა** — `prefers-reduced-motion` მხარდაჭერა, ARIA
- **პროგრესის განულების** ღილაკი

## 🛠 ტექნოლოგია

სუფთა **vanilla JavaScript, HTML, CSS** — build-ის, დამოკიდებულებების და framework-ის გარეშე.

```
computer-from-zero/
├── index.html          # ერთადერთი გვერდი, ტვირთავს ყველა გაკვეთილს
├── app.js              # ბირთვი: ვიჯეტები (CFZ), ნავიგაცია, პროგრესი
├── style.css           # ყველა სტილი + მუქი თემა
├── serve.py            # ქეშ-გამორთული dev სერვერი
└── lessons/            # თითო ფაილი = ერთი გაკვეთილი
    ├── electricity.js
    ├── binary.js
    └── ...
```

ახალი გაკვეთილის დასამატებლად: შექმენი `lessons/xxx.js` (გამოიყენე `window.CFZ` ვიჯეტები) და დაამატე `<script>` `index.html`-ში სასურველ ადგილას.
